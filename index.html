Support – Svenska

Vad är Västtrafik Widget?

Västtrafik Widget är en iOS-app som visar realtidsavgångar från Västtrafiks kollektivtrafik direkt på din hemskärm eller låsskärm. Du väljer vilka hållplatser du vill följa och appen hämtar automatiskt aktuella avgångstider.

Hur använder jag appen?

  • Öppna appen och sök efter din hållplats via namn.
  • Lägg till upp till 5 hållplatser.
  • Lägg till widgeten på hemskärmen (håll in på hemskärmen → trycka på + → välj Västtrafik Widget).
  • Filtrera trafikslag och specifika linjer per hållplats via inställningsikonen.

EN

Support – English

What is Västtrafik Widget?

Västtrafik Widget is an iOS app that shows real-time departures from Västtrafik public transit directly on your home screen or lock screen. Add up to 5 stops and the widget updates automatically with live departure times.

How do I use it?

  • Open the app and search for your stop by name.
  • Add up to 5 stops.
  • Long-press your home screen → tap + → choose Västtrafik Widget.
  • Filter by transport type or specific lines using the settings icon next to each stop.

Privacy Policy

Privacy Policy

Last updated: March 2026

No personal data collected

Västtrafik Widget does not collect, store, or transmit any personal data. There are no user accounts, no registration, and no login required.

Data stored on your device

The only data stored is your stop preferences (stop names, IDs, and filter settings), saved locally on your device in a JSON file shared between the app and the widget via iOS App Groups. This data never leaves your device.

API requests

The app communicates with Västtrafik's public transit API (ext-api.vasttrafik.se) to fetch departure times. These requests contain only stop identifiers (public IDs) — no personal information is included. An OAuth 2.0 access token is used for authentication and is stored securely in the iOS Keychain.

No analytics or tracking

The app contains no analytics, crash reporting, advertising, or third-party SDKs of any kind.

Children's privacy

The app does not knowingly collect data from anyone, including children under 13.

Contact

Questions about privacy? Open an issue at github.com/ChrisGbg/VasttrafikWidget/issues

API

Data source – Västtrafik API

All transit data is fetched from the official Västtrafik API Planera Resa v4, operated by Västtrafik (Västra Götalandsregionen).

Endpoints used

  • GET /pr/v4/locations/by-text — stop search by name
  • GET /pr/v4/stop-areas/{gid}/departures — real-time departures for a stop

Authentication

The app uses OAuth 2.0 Client Credentials flow. An access token is obtained from ext-api.vasttrafik.se/token and cached in the iOS Keychain for its validity period (typically 3 600 seconds).

API documentation

github.com/vasttrafik/api-pr-docs — API Planera Resa v4 documentation
